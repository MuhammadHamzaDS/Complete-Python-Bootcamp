# Complete Python Bootcamp 🐍

Learn Python from **basics to projects** in one complete bootcamp.
Perfect for beginners who want to build strong programming skills.

---

## Features

* Python fundamentals: variables, data types, loops, conditionals
* Functions, modules, and Object-Oriented Programming
* File handling and error management
* Hands-on exercises and small projects

---

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/MuhammadHamzaDS/Complete-Python-Bootcamp.git
```

2. Navigate to the folder:

```bash
cd Complete-Python-Bootcamp
```

3. Run Python scripts:

```bash
python filename.py
```

> Make sure you have **Python 3.x** installed.
