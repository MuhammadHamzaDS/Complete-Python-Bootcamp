# 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/MuhammadHamzaDS/Complete-Python-Bootcamp.git
```

### 2️⃣ Move into the Project Directory

```bash
cd Complete-Python-Bootcamp
```

### 3️⃣ Run Any Python File

```bash
python filename.py
```

> **Make sure Python 3.x is installed on your system.**
